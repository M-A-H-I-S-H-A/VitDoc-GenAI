VitDoc-GenAI

🔗 Live Demo: https://vitdoc-genai-production.up.railway.app/

📌 Overview

VitDoc-GenAI is a web application that uses a Large Language Model (LLM) to process and generate intelligent responses for user inputs. It enhances traditional systems by providing smart, AI-driven outputs.

🚀 Features
LLM-based response generation
User-friendly interface
Fast and interactive results
Backend API integration

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
AI: Gemini API (LLM)

⚙️ How to Run Locally
git clone https://github.com/your-username/VitDoc-GenAI.git
cd VitDoc-GenAI
npm install
node server.js

📊 Comparison

Compared to the non-LLM version:
More intelligent responses
Better user interaction
Higher flexibility

🔐 API Note
This project uses the Gemini API key for generating AI responses.
Make sure to add your API key in environment variables before running locally.

📎 Notes

This is the LLM-based version of the project. A non-LLM version is implemented separately for comparison.
